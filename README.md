# YouTube Speed Controller

A lightweight Chrome/Brave extension that adds more playback speed options to YouTube.

## Features
- Adds **2.5x, 3x, 3.5x, and 4x** playback speeds to the YouTube video player.
- Integrates directly into the native YouTube "Playback speed" settings menu.
- Works seamlessly with the existing YouTube interface.

## Installation

### From Source (Developer Mode)
1.  Clone this repository or download the source code.
2.  Open your browser (Chrome or Brave).
3.  Navigate to the Extensions page:
    -   **Chrome**: `chrome://extensions`
    -   **Brave**: `brave://extensions`
4.  Enable **Developer mode** in the top right corner.
5.  Click **Load unpacked**.
6.  Select the folder containing this extension's files.

## Usage
1.  Open any YouTube video.
2.  Click the **Settings (gear icon)** in the video player.
3.  Select **Playback speed**.
4.  Choose one of the new high-speed options (2.5x - 4x).
